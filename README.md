**Clarification: This repository has build scripts, configurations, and core structure for Verlian operating system. Components and packages are built in C and C++**

**What's Verlian?**
Verlian is a debian-based distro that replicates the Arch Linux experience, this distro is mostly educational and for experimentation. It’s also ideal for servers and development environments for those who need repositories compatible with Debian's.

**What’s Verlian’s philosophy:** 
its main philosophy is "whatever you need" Although Verlian is deeply inspired in Kiss (Keep It Simple Stupid)
